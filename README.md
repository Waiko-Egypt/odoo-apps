# Odoo Free Modules Collection

Welcome to the **Odoo Free Modules** repository!  
This repository contains a collection of high-quality, open-source modules for [Odoo](https://www.odoo.com/), designed to extend and enhance the functionality of your Odoo instance — **completely free of charge**.

---

## 📦 What's Inside

This repository includes a variety of custom Odoo modules that are:

- ✅ **Free to use** – No licensing fees, no hidden costs.
- ✅ **Open source** – Licensed under LGPL / MIT / OPL (see each module for details).
- ✅ **Community-driven** – Built with love by Odoo developers and contributors.
- ✅ **Regularly updated** – Compatible with Odoo versions **13.0 through 19.0**.

---

## 🚀 Getting Started

### Prerequisites

- Odoo (version 13.0 or higher recommended)
- Python 3.6+ (Python 3.10+ recommended for newer versions)
- PostgreSQL 10+
